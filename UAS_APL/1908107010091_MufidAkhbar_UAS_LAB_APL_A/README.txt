Pada kasus MobilePhone ini, saya membuat code ini seperti adanya penjualan MobilePhone. Yang mana penjualan ini berisikan MobilePhone berbagai Merk dan Harga, lalu user akan memilih. Pada Strategy, saya membuat Jenis-Jenis Merk MobilePhone ini beserta Informasi Harganya. User dapat memilih MobilePhone yang mana diinginkan. Lalu Harga yang tertera disetiap MobilePhone akan di Adapterkan dari Dollar Menjadi Rupiah.
Jadi Kesimpulan nya adalah, saya Implementasikan 2 pattern ini menjadi 1 kasus yaitu:
Strategy Pattern: Menawarkan Berbagai Mobile Phone
Adapter Pattern: Mengubah Mata Uang Dollar Menjadi Rupiah (kurs 14.000)

Saya Menggabungkan Tugas Strategy dan Adapter pada tugas sebelumnya lalu saya sesuaikan pada Main.java nya
